An example of <seed-element>:

Everything that has a darkviolet background is part of the demo/index.html page.

Call a method on the seed-element with document.querySelector('seed-element').sayHelloTo('World') and output the return value with alert().

Call seedElement.triggerSomeEvent() which fires an event internally from the seed-element. We've registered a listener for that event which will print out some details from the event to the console.

This is the content block of the <seed-element> in demo/index.html It is referenced in the web component as <component></component>.
